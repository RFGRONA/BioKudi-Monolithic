# BioKudi

[![Build and deploy ASP.Net Core app to Azure Web App - biokudi](https://github.com/RFGRONA/BioKudi/actions/workflows/master_biokudi.yml/badge.svg?branch=master)](https://github.com/RFGRONA/BioKudi/actions/workflows/master_biokudi.yml)

La falta de una plataforma dedicada al turismo ecológico en Cundinamarca dificulta la experiencia de viaje para quienes buscan opciones sostenibles. La carencia de información actualizada sobre destinos ecoturísticos dificulta la planificación y disfrute de actividades en entornos naturales y comunidades locales. Para abordar este desafío, Biokudi ofrece una solución innovadora.

## Características Principales:

- **Gestión de usuarios por rol:** Para una experiencia personalizada.
- **Creación de cuentas:** Para usuarios individuales y administradores.
- **Manejo completo de operaciones CRUD:** Por parte de los administrativos para mantener la plataforma actualizada.
- **Mapa interactivo:** Que permite explorar destinos ecoturísticos y obtener información detallada.
- **Creación de reseñas de lugares:** Para compartir experiencias y recomendaciones.
- **Sistema de tickets:** Para gestionar consultas y problemas de los usuarios.
- **Generación de reportes PDF:** Para análisis y seguimiento.
- **Notificaciones por correo:** Para mantener a los usuarios informados.

## Requisitos del Proyecto:

- Utiliza **.NET Core 8** y **SQL Server** para una base sólida.
- Los archivos necesarios para la base de datos se encuentran en la carpeta `Data`.
- Proyecto académico en fase **beta**, en constante desarrollo y mejora.
